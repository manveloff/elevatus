# Вводные
- Даны две строки a и b.
- Пусть ДЛИНА РЕДАКТИРОВАНИЯ (ДР) между a и b - минимальное
    кол-во операций, необходимых для превращения a в b.

# Задание

    Напишите php-код, который вычисляет ДР, следуя одному
    из следующих правил:

    1. Метод вычисления расстояния Хэмминга
    - Можно заменять одну букву другой.
    - Можно заполнять более короткий текст пробелами.

    2. Метод вычисления расстояния Левинштейна
    - Есть 3 операции: вставить, удалить, заменить.

    Таким образом, соответствующий функции должны возвращать
    число int с наименьшим возможным числом шагов:

        hamming_dis(a, b) returns integer
        levenshtein_dis(a, b) returns integer

# Требования к коду
- ✓ OOP
- ✓ Отдельный класс для каждого из методов (Хэмминга / Левинштейна).
- ✓ В каждом классе д.б. 1 static public метод, который
    создаёт объект, инициализирует состояние, выполняет
    вычисления и возвращает результат.
- ✓ Можно добавлять private/protected свойства по мере
    необходимости для входящих и промежуточных данных.
- ✓ Private/Pretected методы, которые можно вызывать
    рекурсивно, если необходимо.
- ✓ Эти методы должны иметь смещение с начала входящих
    текстов вместо подстановки подстрок.
- ✓ Документация и коменты приветствуются.
- ✓ Напишите другой файл, который require ваш файл,
    и проводит какие-то тесты на нём.
- ✓ Напишите CLI который запрашивает 2 значения и 
    отображает дистанцию Левинштейна.
- ✓ Сделатйте WEB-страницу с формой из 2-х полей,
    и возвращающую дистанцию Левинштейна.
- В документации опишите, как запустить встроенный в 
  PHP веб-сервер для просмотра этой страницы.    

# Примеры вычисления дистанции Левинштейна

    a: "this is a test"
    b: "this is test"
    levenshtein: 2 operations (remove a and next space)    

    a: "this is test"
    b: "the is test"
    levenshtein: 2 operations (replace i with e in this and remove s)
